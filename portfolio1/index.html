Recent Projects

Project 1

Concrete Strength Prediction

In my first project, I developed a machine learning model to predict concrete compressive strength. Using features like cement, water, and aggregates proportions, I applied advanced algorithms for data analysis, preprocessing, feature selection, and model training. Techniques included regression analysis and cross-validation for accuracy. This project provides valuable insights for optimizing concrete mixes in construction, enhancing decision-making and structural requirements.

Project 2

Multimodal Thalassemia Prediction

In my second project, I focused on predicting thalassemia using multimodal data and deep learning techniques. This project involved integrating various types of data, including genetic information, blood test results, and medical imaging, to build a comprehensive predictive model. By leveraging deep learning algorithms, I was able to effectively analyze complex patterns and relationships within the data, leading to more accurate predictions of thalassemia. This project demonstrates the potential of advanced AI techniques in enhancing medical diagnostics and providing better outcomes for patients through early and precise detection of genetic disorders.
